# 🍎 Fruit Image Classifier (CNN)

## 🚀 Overview
This project uses a Convolutional Neural Network (CNN) to classify fruit images into categories like apple, banana, and orange.

## 🧠 Model
- CNN with 3 convolution layers
- MaxPooling layers
- Dense fully connected layers

## 📊 Dataset
- Custom dataset or Fruits-360
- Images resized to 100x100

## ▶️ Run Project

```bash
pip install -r requirements.txt
python src/train.py
python src/predict.py
